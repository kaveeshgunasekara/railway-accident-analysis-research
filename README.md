#  Railway Accident Analysis

## Overview
This project analyses railway accident data to identify key patterns and risk factors using data analytics techniques.

## Dataset
Public railway accident dataset (originally sourced from Kaggle).  
A sample dataset is included due to size limitations.

## Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Plotly (Graph Objects)
